# job4j_cinema

#Кинотеатр - Система покупки билетов
#1. Описание:
- Веб-приложение для онлайн-покупки билетов в кинотеатр. Пользователи могут просматривать расписание сеансов, изучать каталог фильмов и бронировать места в зале.

#2. Основной функционал 
- Публичный доступ: просмотр расписания сеансов и каталога фильмов
- Регистрация и вход для пользователей
- Покупка билетов
- Защита от двойного бронирования одного места

#3. Технологии
- Backend: Spring Boot, Spring Security, Spring MVC
- Frontend: Thymeleaf, Bootstrap 5
- База данных: PostgreSQL
- Миграции: Liquibase
- Доступ к данным: Sql2o

#4. Страницы приложения
- / — главная страница
- /sessions — расписание сеансов
- /films — каталог фильмов
- /tickets/{id}/buy — покупка билета
- /register — регистрация
- /login — вход в систему
