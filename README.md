# job4j_cinema

Веб-приложение «Кинотеатр» — система онлайн-покупки билетов.

## Описание
Пользователи могут просматривать расписание сеансов, изучать каталог фильмов и бронировать места в зале. 

## Основной функционал
- Публичный доступ: просмотр расписания сеансов и каталога фильмов. 
- Регистрация и вход для пользователей. 
- Покупка билетов. 
- Защита от двойного бронирования одного места. 

## Технологии
- Backend: Spring Boot, Spring Security, Spring MVC. 
- Frontend: Thymeleaf, Bootstrap 5. 
- База данных: PostgreSQL. 
- Миграции: Liquibase. 
- Доступ к данным: Sql2o. 

## Страницы приложения
- `/` — главная страница. 
- `/sessions` — расписание сеансов. 
- `/films` — каталог фильмов. 
- `/register` — регистрация. 
- `/login` — вход в систему.
